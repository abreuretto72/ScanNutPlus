üß† 1. The Artificial Intelligence Engine (RAG & Gemini)

The core of the application beats through Google Gemini Pro linked to a RAG (Retrieval-Augmented Generation) architecture. It doesn't just "reply" like ChatGPT; it evaluates the patient.

  • Biometric Identity and Context: The AI knows exactly who it is analyzing. Before issuing any report, it injects the animal's systemic variables (Breed, Sex, Age, Dietary Restrictions) via the local database (ObjectBox).
  • Zero Hallucinations (Scientific Sources): RAG is strictly grounded. If the AI receives a photo of dermatitis or a food sample, it is expressly prompted to append Scientific Sources and References at the end of the report (e.g., Merck Veterinary Manual).
  • Urgent Data Handling: The API does not return raw text. It returns JSON data structured by the Master Protocol 2026 containing an URGENCY tag (Green, Yellow, Red). This tag colors the entire mobile screen in real time.

ü©∫ 2. Multimodal Diagnostic Capabilities

The UniversalAiService is the "clinic" compressed into the hardware. The user opens the camera and selects one of the operational specialties:

  • Dermatology: Photographing hair loss (alopecia) or redness.
  • Dentistry: Analysis of gums to detect tartar plaque and gingival retraction.
  • Gastroenterology (Stool/Urine): Evaluation through the Bristol Stool Chart.
  • Kinesiology / Orthopedics: Triage through Body Condition Score via lateral photographs.
  • Behavioral and Vocal Biology: Active processing of videos to identify pain/aggression or native AUDIO files for the AI to discover reasons for recurrent coughs, barks, or wheezing.
  • Analytical Labs (OCR): The app scans physical blood test papers, digitizes via Gemini, crosses with reference values, and builds an executive summary for laymen.
